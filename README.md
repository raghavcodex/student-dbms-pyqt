# 🎓 Student Database Management System (PyQt5 + SQLite)

A desktop-based Student Database Management System built using **Python (PyQt5)** and **SQLite3**.
It provides a simple GUI to manage student records with basic CRUD operations.

---

## 📌 Project Overview

This project allows users to manage student data efficiently through a graphical interface.
It is designed for academic purposes to demonstrate database management using Python.

---

## 🚀 Features

* ➕ Add new student records
* 🔍 Search student by Roll Number
* ❌ Delete student record
* 🗄️ SQLite database integration
* 🖥️ GUI built using PyQt5

---

## 🛠️ Tech Stack

* Python 3.x
* PyQt5 (GUI Framework)
* SQLite3 (Database)

---

## 📁 Project Structure

```
sdbms.py        → Main application file  
user.png        → UI image  
sdms.db         → Auto-generated database file (ignored in GitHub)  
README.md       → Project documentation  
```

## ▶️ How to Run the Project

### Step 1: Install Python dependencies

```bash
pip install PyQt5
```

---

### Step 2: Run the application

```bash
python sdbms.py
```
