# 🎓 Student Database Management System (SQLite + PyQt5)

## 📌 Project Description
A desktop-based Student Database Management System built using Python (PyQt5) and SQLite.  
It supports basic CRUD operations like Add, Search, and Delete students.

---

## 🚀 Features
- Add Student Record
- Search Student by Roll Number
- Delete Student Record
- SQLite Database Integration
- GUI built with PyQt5

---

## 🖥️ Tech Stack
- Python 3.x
- PyQt5
- SQLite3

---

## 📁 Project Structure
- `sdbms.py` → Main application file
- `user.png` → UI image
- `sdms.db` → Auto-generated database file
